# StatsLand

Below you can find minimal information on how to run the application.

## Installation

```bash
pip3 install -r requirements.txt
```

## Usage

```bash
flask --app app run --debug
```

*Note.* Make sure the `OPENAI_API_KEY` environment variable is set.
